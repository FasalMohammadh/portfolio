## Getting Started

---

- Prerequisites

  - [Node](https://nodejs.org/en/)

- Installation

  - Clone the repo

    ```sh
    git clone git@github.com:FasalMohammadh/portfolio.git
    ```

  - Install NPM packages

    ```sh
    npm install
    ```

  - Start the server
    ```sh
    npm run dev
    ```
